Selama ini, saat mengerjakan project menggunakan react kita terbiasa mengimport file seperti ini `../../../Components/Dashboard/Profil` dan tidak ada yang salah dengan melakukannya seperti ini. Namun mengelola semua `../` ini cukup sulit dan kadang membuat error saat memanggil class atau function yang telah dibuat. Katakanlah saya ingin mengubah lokasi file atau folder. Dengan melakukan import menggunakan relative path kita perlu mengetahui posisi file tersebut relatif terhadap file yang mengimportnya. Jadi Relative Path sangat tergantung pada struktur folder yang kita buat

<br />
Di sini datang absolute import sebagai solusi, Sekarang kita dapat mengimpor file
dalam bentuk ini `Components/Dashboard/Profil` tanpa repot dengan `../` dan kita
tidak pelru memikirkan tentang posisi relatif dari dua file tersebut. Yang kita pedulikan
hanya posisi file tersebut relatif terhadap root dari proyek. Ini mempermudah pekerjaan
mengimpor banyak.

<br />
## Keuntungan dari absolute import :

<ol>
  <li>
    Dengan absolute import, Anda tidak perlu memikirkan posisi relatif dari file
    yang ingin Anda import. Anda hanya perlu mengetahui posisi file tersebut
    relatif terhadap root proyek, sehingga lebih mudah untuk memelihara dan
    mengubah struktur proyek.
  </li>
  <li>
    Lebih mudah melakukan debugging karena Anda tidak perlu mengikuti alur `../`
    untuk menemukan file yang diimport. Anda hanya perlu melihat lokasi file
    tersebut relatif terhadap root proyek.
  </li>
  <li>
    Membuat kode terlihat lebih rapi dan lebih mudah dibaca karena tidak perlu
    memikirkan relasi antara file yang diimport dan file yang mengimportnya. Ini
    membuat kode Anda lebih mudah dipahami oleh orang lain yang bekerja di
    proyek yang sama.
  </li>
</ol>

<br />

Dengan menggunakan absolute import, kita dapat mengimpor file dengan mudah dan lebih mudah untuk mengelola proyek kita. Mari lihat bagaimana cara pasti untuk melakukannya.üòÅ

## Get Started

Menurut dokumen create-react-app , Kita dapat menggunakan absolute import dalam proyek react kita dengan mengonfigurasi file `jsconfig.json / tsconfig.json` (untuk proyek TypeScript) di root proyek kita. Jika Anda menggunakan TypeScript di proyek Anda, Anda sudah memiliki file `tsconfig.json`.

Jika bekerja di **JS**, tambahkan file `jsconfig.json` di root proyek Anda dengan konten berikut:

<div className="code-container">
```
{
  "compilerOptions": {
    "baseUrl": "src"
  },
  "include": ["src"]
}
```
</div>

Jika Anda menggunakan **TypeScript**, Anda sudah memiliki file `tsconfig.json` di root proyek Anda. Anda dapat mengonfigurasi pengaturan `baseUrl` di dalam `compilerOptions` file `tsconfig.json` proyek Anda untuk mulai bekerja dengan impor absolut. Konten file akan terlihat seperti ini:

<div className="code-container">
```
{
  "compilerOptions": {
    ...,
    "baseUrl": "src"
  },
  "include": [
    "src"
  ]
}
```
</div>

Dalam file di atas, saya baru saja menambahkan pengaturan `baseUrl` dengan nilai src di `compilerOptions` saya. Tidak ada lagi yang diubah dari `tsconfig.json` asli yang dihasilkan.

## Converting Relative Imports into Absolute Imports

Kami telah mengonfigurasi proyek kami untuk menggunakan impor absolut, sekarang mari kita gunakan.

Tanpa menggunakan absolute imports

<div className="code-container">
```
import ContentWrapper from '../../../../Components/ContentWrapper'

const SampleComponent = () => {

return (

<ContentWrapper
  style={{
    width: '100%',
    padding: '20px 10px',
    marginBottom: '10px',
  }}
>
  <div>This is a sample component.</div>
</ContentWrapper>
) }

export default SampleComponent
```
</div>

Sekarang, kita dapat menghilangkan `../../../` dari path import. Setelah itu filenya akan terlihat seperti ini:

<div className="code-container">
```
import ContentWrapper from 'Components/ContentWrapper'

const SampleComponent = () => {

return (

<ContentWrapper
  style={{
    width: '100%',
    padding: '20px 10px',
    marginBottom: '10px',
  }}
>
  <div>This is a sample component.</div>
</ContentWrapper>
) }

export default SampleComponent
```
</div>

<br />
Sekarang setelah kami mengonfigurasi proyek kami untuk mendukung absolute import,
kami dapat meng-import modul yang terletak di `src/components/Button.js`, seperti:
<br />
`import Button from 'components/Button';`
